=== FULL END-TO-END PIPELINE MAP ===

FRACFOCUS PIPELINE (monthly cron + manual trigger):
  POST /api/v1/sync/trigger
    -> SyncService.run_sync()
      -> _do_sync()
        -> DownloadService.check_remote_changed()      [HEAD request, ETag/Last-Modified]
        -> DownloadService.read_zip_csv_infos()         [ZIP metadata only]
        -> CsvFileStateRepository.get_changed_files()
        -> DownloadService.extract_files()
        -> CsvIngestionService.process_csv() x N CSVs
          -> infer_columns()                            [header normalization]
          -> FracFocusRepository.create_table_if_not_exists()
          -> FracFocusRepository.ensure_columns()
          -> FracFocusRepository.replace_csv_data()     [atomic DELETE+INSERT]
        -> SyncStateRepository.upsert()                 [etag written last]
        -> SyncHistoryRepository.finish()

TEXNET PIPELINE:
  POST /api/v1/seismic/texnet/fetch
    -> TexNetService.fetch_delaware_events()
      -> _query_page() x N (paginated ArcGIS REST)
      -> _normalize()
    -> SeismicEventRepository.upsert_many()  [source="texnet"]
    -> SyncHistoryRepository.finish()

USGS PIPELINE:
  POST /api/v1/seismic/usgs/fetch
    -> USGSService.fetch_delaware_events()
      -> _query_page() x N (paginated FDSN)
      -> _normalize()
    -> SeismicEventRepository.upsert_many()  [source="usgs"]
    -> SyncHistoryRepository.finish()

IRIS PIPELINE:
  POST /api/v1/seismic/iris/stations/fetch
    -> IRISService.fetch_delaware_stations()  [pipe-delimited text]
      -> _normalize()
    -> IRISStationRepository.upsert_many()

SWD UIC PIPELINE (checkpoint-resumable):
  POST /api/v1/swd/uic/fetch
    -> UICService.fetch_delaware_wells()
      -> Socrata paginated API x N pages
      -> _normalize()
    -> SWDRepository.upsert_wells()
    -> SWDRepository.save_checkpoint()

SWD H-10 PIPELINE (run after uic/fetch; checkpoint-resumable):
  POST /api/v1/swd/h10/fetch
    -> SWDRepository.get_all_uic_numbers()
    -> H10Service.fetch_for_wells()
      -> _fetch_chunk() x N chunks
      -> _normalize()
    -> SWDRepository.upsert_monitoring()
    -> SWDRepository.save_checkpoint()

ANALYSIS PIPELINE:
  GET /api/v1/analysis/events/{event_id}/context  [read-only, no snapshot]
    -> EventContextService.assemble()
      -> SeismicEventRepository.get_by_event_id()
      -> SWDRepository.find_wells_in_bbox() -> haversine_km filter
        -> SWDRepository.get_monitoring_window() per well
      -> FracFocusRepository.find_nearby()
      -> IRISStationRepository.find_stations_in_bbox() -> haversine_km filter

  POST /api/v1/analysis/events/{event_id}/analyze  [runs attribution + persists snapshot]
    -> EventContextService.assemble()               [same as above]
    -> PhysicsAttributionService.score()
      -> _swd_score()    [erfc pore-pressure diffusion]
      -> _frac_score()   [spatial decay]
      -> MonteCarloFracSampler.sample()  [2000 trials -> (mean, p5, p95)]
      -> _verdict()      [log-odds/softmax -> (driver, confidence)]
    -> EventContextRepository.save_snapshot()

TWO SQLALCHEMY PARADIGMS:
  fracfocus table    -> SQLAlchemy Core  (dynamic schema from CSV header)
  all other tables   -> SQLAlchemy ORM   (fixed schemas with _ensure_*_columns migration)
